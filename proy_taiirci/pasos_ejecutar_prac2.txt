PASOS PARA EJECUTAR LA PRÁCTICA 3.1 (PRÁCTICA 2)
PARTIENDO DE LA INFRAESTRUCTURA DE LA PRÁCTICA 1 YA CONFIGURADA

TERMINAL 1 — Levantar la infraestructura Docker
cd /opt/aq-iiot-lab
docker compose up -d

# Verificar que están corriendo mosquitto, influxdb, grafana y telegraf:
docker ps

# Si telegraf aparece como Exited, es porque el servidor OPC-UA no está levantado todavía.
# Levantarlo después del paso TERMINAL 2 con:
docker compose restart telegraf


TERMINAL 2 — Servidor OPC-UA
cd /opt/aq-iiot-lab/python
source venv/bin/activate
python3 opcua_server_bridge.py

# Una vez arrancado, volver al TERMINAL 1 y hacer:
docker compose restart telegraf


TERMINAL 3 — Intermediario
cd /opt/aq-iiot-lab/python
